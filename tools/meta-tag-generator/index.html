メタタグジェネレータ

charset

文字コードを設定するメタタグです！

文字コード

生成コード

<meta charset="UTF-8">

description

検索結果の下に表示されるサイト説明の文章です！

サイトの説明文

生成コード

<meta name="description" content="ここに説明文を入力します！">

viewport

表示領域の幅を決めるタグです！

表示幅(width)

初期倍率(initial-scale)

ズーム操作の許可(user-scalable)

生成コード

<meta name="viewport" content="width=device-width, initial-scale=1, user-scalable=yes">

theme-color

テーマカラーを設定するタグです！

カラーコード

選択：#000

生成コード

<meta name="theme-color" content="#000">

og

OG設定するタグです！

og:site_name

og:url

og:type

og:title

og:description

og:image

生成コード

                        <meta property="og:site_name" content="サイト名を入力します！">
<meta property="og:site_name" content="サイト名を入力します！">
<meta property="og:url" content="ページのURLを入力します！">
<meta property="og:type" content="ページの種類を入力します！">
<meta property="og:title" content="ページのタイトルを入力します！">
<meta property="og:description" content="ページの説明を入力します！">
<meta property="og:image" content="ページのサムネイル画像のURLを入力します！">

robots

検索エンジンのクローラの動きを設定するメタタグです！

ディレクティブ

生成コード

<meta name="robots" content="all">

keywords

Webページに関連する情報を伝えるタグです！

キーワード設定(例:キーワード1,キーワード2,キーワード3)

生成コード

<meta name="keywords" content="キーワード1,キーワード2,キーワード3">

最終的なコードは以下に出力されます！
